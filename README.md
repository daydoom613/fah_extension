# Fahhh Sound 🚨

Plays a sound instantly whenever your code crashes or a terminal execution fails. 

## Features
* **Instant Feedback:** Uses native background processes for zero-delay audio playback.
* **Language Agnostic:** Works perfectly with Python, JavaScript, C++, or any terminal command that throws a non-zero exit code.
* **Stealthy:** Runs entirely in the background with absolutely no media player popups.

## How it Works
Just run your code normally. If it fails, you hear "Fahhh". If it succeeds, silence. 

**Enjoy the most stressful debugging experience of your life!**
